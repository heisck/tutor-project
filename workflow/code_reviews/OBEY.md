# Code Reviews

Code review notes and feedback documents.

## What Goes Here

- Code review feedback
- Review checklists
- Quality gate reports
- Security audit findings

## Structure

```
code_reviews/
├── <review>/          # Active reviews at root
├── templates/         # Review templates
└── dungeon/           # Archived reviews
    ├── completed/
    ├── archived/
    └── someday/
```

## Guidelines

- Link reviews to PRs/issues
- Track action items from reviews
- Use `camp dungeon crawl` to triage completed reviews into dungeon
