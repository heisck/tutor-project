# HUMAN-IN-THE-LOOP UI MODE (MANDATORY)

For all UI-related tasks:

## DO NOT
- Do not design UI independently
- Do not choose colors or layout on your own
- Do not generate full pages without user direction

## DO
1. Ask the user for:
   - layout structure
   - component references (images, links, ideas)
   - design direction
   - animation preferences

2. Present options:
   - layout options
   - component choices
   - interaction styles

3. Wait for confirmation before implementing

4. Use real UI systems:
   - Tailwind CSS
   - shadcn/ui
   - Magic UI
   - React Bits
   - 21st.dev references

5. Build UI like LEGO:
   - assemble components
   - reuse patterns
   - avoid inventing layouts

6. Ask before:
   - adding animations
   - choosing spacing
   - applying visual styling

## GOAL
The human controls design.
The AI executes implementation.
