# Intent Packet

## Core Intent

The system must transform uploaded learning materials into a structured, adaptive tutoring experience that teaches concepts step by step and verifies real understanding.

## Key Objectives

- Turn uploaded learning materials into structured knowledge units
- Preserve all meaningful learning content
- Organize ideas into concepts and prerequisite relationships
- Teach one concept at a time in a guided sequence
- Adapt explanations based on learner responses
- Detect confusion early and respond appropriately
- Require evidence before marking mastery
- Allow exact session pause and resume
- Answer learner questions using document-grounded context only

## Non-Goals

- The system is not a summarizer
- The system must not skip concepts silently
- The system must not answer outside the uploaded document context
- The system must not assume understanding without evidence

## Intent Constraints

- Teaching must prioritize clarity before formalism
- Progression must depend on demonstrated understanding
- Retrieval must remain scoped to the current user and document
- The experience must adapt to the learner, not use a fixed script
