# Purpose

## System Goal

Build an adaptive AI tutoring system that transforms uploaded learning materials into a guided, interactive teaching experience.

## What Done Means

A student can:
- Upload a document
- Receive structured teaching instead of summary-only output
- Learn concept by concept
- Be checked for real understanding
- Pause and resume from the exact point
- Ask grounded questions at any time

## Why This Matters

Most tools summarize content. This system must teach it.

It should:
- Preserve all meaningful ideas
- Detect misunderstanding early
- Adapt explanations to the learner
- Require evidence before mastery
- Keep all answers grounded in the uploaded material

## Core Principles

- Story-first explanation
- Surface-first understanding
- No fake mastery
- No silent skipping
- Document-grounded teaching
